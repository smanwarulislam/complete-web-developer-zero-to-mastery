Resources: Public APIs

Once you learn about APIs, as you have just done, you realize that there is a world of possibilities for what you can build.
Your own startup? Your own app? The possibilities are endless.
Here is my favorite resource listing some of the Public APIs you can use to build anything you want. Enjoy exploring it!

A Collaborative List Of 1400+ Public APIs For Developers
https://publicapis.dev/

public-apis/public-apis: A collective list of free APIs
https://github.com/public-apis/public-apis

Try playing this game, read the documentation, and use the developer tools to "debug".
It will be good practice and a lot of fun!
